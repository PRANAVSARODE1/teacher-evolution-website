Troubleshooting

Microphone Permission Issues:

  • Chrome/Edge: Click the lock icon in the address bar → Allow microphone access
  • Firefox: Click the shield icon → Allow microphone permissions
  • Safari: Safari → Preferences → Websites → Microphone → Allow

Common Solutions:

  • Ensure your microphone is connected and working
  • Close other applications that might be using the microphone
  • Check your system's privacy settings for microphone access
  • Try refreshing the page and allowing permissions again
  • Make sure you're using HTTPS (required for microphone access)
